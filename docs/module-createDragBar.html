createDragBar

Factory function for creating draggable bar elements that appear above each chart. Drag bars display a compact list of channel colors and labels, allowing users to visually identify chart contents and drag charts to reorder them. Key Features: - Displays colored squares (one per channel) for visual identification - Shows channel labels next to each color indicator - Supports drag-and-drop via HTML5 Drag API - Reactively updates labels when channelState changes - Handles both analog and digital channel types - Subscribes to yLabels updates for live synchronization - Graceful fallback if subscription fails DOM Structure: - Container:
- Rows:
containing color span + label span - Color: with background-color CSS - Label: Dependencies: - helpers.js: createCustomElement utility - channelState: Reactive state with subscribe method - cfg: COMTRADE config with channel metadata

# inner createDragBar(group, cfg, channelState) → {HTMLDivElement}

Create a draggable color bar displaying channel identities for a chart. Builds a visual drag handle element containing: - Color squares corresponding to each channel in the chart - Channel labels next to each color - Reactive subscriptions to update labels when state changes The drag bar serves multiple purposes: 1. Visual identification of which channels are in each chart 2. Drag handle for reordering charts (via setupChartDragAndDrop) 3. Live update target when channel labels change in UI Internal Behavior: - Infers channel type (analog/digital) based on index ranges - Attempts to subscribe to yLabels updates via createState API - Handles both whole-array and single-element updates - Gracefully ignores subscription errors on fallback systems - Preserves data-global-index attributes for live label updates
Parameters:
Name Type Attributes Description
group Object Group/chart definition object
indices Array.<number> Global channel indices displayed in this chart
colors Array.<string> Color hex codes (currently unused, uses channelState instead)
cfg Object COMTRADE configuration object
analogChannels Array.<Object> <optional>
 Analog channel metadata
digitalChannels Array.<Object> <optional>
 Digital channel metadata
channelState Object Reactive channel state (createState proxy)
analog Object Analog channel state
analog.yLabels Array.<string> Channel names/IDs
analog.lineColors Array.<string> Hex color codes
digital Object Digital channel state (same structure)
subscribe function Subscription method for reactive updates

// Create drag bar for analog chart with 3 channels
const dragBar = createDragBar(
  { indices: [0, 1, 2], colors: ['#FF0000', '#00FF00', '#0000FF'] },
  cfg,
  channelState
);

// HTML output structure:
// <div class="dragBar" draggable="true" style="display:flex;...">
//   <div class="dragBar-row" style="display:flex;...">
//     <span class="dragBar-color" style="background:#FF0000;..."></span>
//     <span class="dragBar-label" data-global-index="0">Phase A Current</span>
//   </div>
//   <div class="dragBar-row">
//     <span class="dragBar-color" style="background:#00FF00;..."></span>
//     <span class="dragBar-label" data-global-index="1">Phase B Current</span>
//   </div>
//   <div class="dragBar-row">
//     <span class="dragBar-color" style="background:#0000FF;..."></span>
//     <span class="dragBar-label" data-global-index="2">Phase C Current</span>
//   </div>
// </div>
// The drag bar automatically updates when labels change
// Before: "Phase A Current"
channelState.analog.yLabels[0] = "Main Feeder A";
// After: Drag bar label automatically updates to "Main Feeder A"
// Digital channel drag bar
const digitalDragBar = createDragBar(
  { indices: [0], colors: ['#FFAA00'] },
  cfg,
  channelState  // Will infer type="digital" based on index ranges
);
