domUpdateQueueInit

Initializes the global DOM update queue and makes it available to createState. Call this once at app startup to enable selectiveUpdate feature in bindToDOM.

View Source utils/domUpdateQueueInit.js, line 1

Example
import { initGlobalDOMUpdateQueue } from './utils/domUpdateQueueInit.js';

// Call this early in your app initialization
initGlobalDOMUpdateQueue();

// Now you can use selectiveUpdate in bindToDOM
state.bindToDOM('prop', el, { selectiveUpdate: true });
