üìê Linear Interpolation & Time Spacing

What Happens Behind the Scenes?
When COMTRADE files are loaded, they may have non-uniform timestamps or multiple sampling rates. Linear interpolation creates uniform time spacing across all samples.
Formula: time = sampleNumber / samplingRate
Example: If sampling rate = 4000 Hz
Sample 0: time = 0 / 4000 = 0.00 seconds
Sample 1: time = 1 / 4000 = 0.00025 seconds
Sample 2: time = 2 / 4000 = 0.0005 seconds
Sample 3: time = 3 / 4000 = 0.00075 seconds
Sample 4: time = 4 / 4000 = 0.001 seconds
